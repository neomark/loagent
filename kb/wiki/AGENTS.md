# 위키 에이전트 지침 (AGENTS.md)

이 파일은 LLM이 위키 지식 베이스를 관리하고 구조화하는 방식을 정의합니다.

## 위키 구조
- `index.md`: 지식 베이스의 전체 개요 및 목차.
- `sources/`: 원본 문서에서 변환된 텍스트 파일들.
- `summaries/`: 각 소스 문서에 대한 요약 페이지.
- `concepts/`: 여러 문서에 걸쳐 나타나는 핵심 개념, 용어, 주제에 대한 합성 페이지.
- `explorations/`: 사용자 질의 및 분석 결과 저장소.

## 생성 규칙
1. 모든 위키 페이지는 한국어로 작성합니다.
2. 문서 간의 연결은 `[[문서명]]` (위키링크) 형식을 사용합니다.
3. 새로운 문서가 추가되면 관련 있는 기존 `concepts/` 페이지를 업데이트하거나 새로운 개념 페이지를 생성합니다.
4. 상충되는 정보가 발견되면 이를 명시하고 플래그를 설정합니다.

## 스타일 가이드
- 명확하고 전문적인 어조를 유지합니다.
- 복잡한 정보는 표나 리스트를 활용하여 구조화합니다.
- 각 페이지 하단에는 관련 페이지 링크를 추가합니다.
